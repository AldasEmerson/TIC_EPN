Nombre del componente:  Control cooperativo de un sistema multi-agente para la resolución de tareas complementarias

Nombre del autor:       Emerson Javier Aldás López

Video 1:                TIC_House_AldasE
Titulo.-                Pruebas y Resultados bajo escenario HOUSE
Descripción.-           Se realiza la ejecución del archivo de lanzamiento MAIN SLAM para la obtención de un mapa bidimensional por parte del Quadrotor en el escenario HOUSE. También se ejecuta el archivo de lanzamiento MAIN AUTO para el desarrollo de la Navegación Autónoma por parte del TurtleBot 3 usando como referencia el mapa obtenido previamente. Como último se realizan pruebas de navegación para garantizar la llegada de robot móvil a sus puntos destino.
